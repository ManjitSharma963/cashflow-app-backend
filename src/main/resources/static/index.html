üìä Available API Endpoints

GET /api/customers

List all customers with optional filtering and pagination

POST /api/customers

Create a new customer

GET /api/customers/{id}

Get customer details by mobile number

PUT /api/customers/{id}

Update existing customer

DELETE /api/customers/{id}

Delete customer (only if no outstanding transactions)

GET /api/transactions

List all transactions with filtering options

POST /api/transactions

Create new transaction (Credit/Payment/Adjustment)

POST /api/payments

Process customer payment

GET /api/dashboard/summary

Get dashboard statistics and analytics

GET /api/categories

Get customer categories

GET /api/settings

Get application settings
